# **App Name**: CampusPreorder

## Core Features:

- Campus Selection & Login: Allow users to select their campus and log in with a verified college ID or email.
- Vendor Browsing: Display on-campus vendors registered under the selected institution.
- Menu Selection with Time Slots: Enable users to select menu items and choose a pickup time slot with real-time availability indicators. Use a tool to determine ideal prep and handoff times for each time slot.
- Integrated Payment: Support checkout with integrated payment options (UPI, cards, wallets).
- Order Confirmation with QR Code: Display an order confirmation with a QR code for pickup.
- Vendor Dashboard: Provide a vendor dashboard for managing menu items, slot capacity, and order batching.
- Kitchen Display Screen (KDS): Show grouped orders per time slot on a kitchen display screen.
- QR Code Scanner: Allow vendors to validate and mark orders as picked up using a QR scanner.

## Style Guidelines:

- Primary color: A vibrant blue (#29ABE2) to reflect trust, security, and modernity, aligning with the campus-verified access.
- Background color: A light, desaturated blue (#E5F6FD), complementing the primary blue while providing a clean and airy feel.
- Accent color: An energetic orange (#F77F00) for action items like 'Order Now' buttons and highlighting available time slots.
- Body and headline font: 'Inter', a sans-serif font for a clean and modern aesthetic.
- Use clear and intuitive icons for menu categories, payment methods, and time slot availability.
- Implement a mobile-first responsive layout that adapts seamlessly to various screen sizes, emphasizing speed and convenience.
- Incorporate subtle animations for transitions and feedback, enhancing user experience without being distracting.
